# 개인정보처리방침

최종 업데이트: 2026-06-10

dkssud(이하 "확장 프로그램")는 사용자의 개인정보를 수집, 저장, 전송하거나 제3자에게 제공하지 않습니다.

## 수집하는 정보

본 확장 프로그램은 다음 정보를 수집하지 않습니다.

- 이름
- 이메일 주소
- 전화번호
- 위치 정보
- 결제 정보
- 방문 기록
- 계정 정보

## 데이터 처리 방식

사용자가 선택한 텍스트는 사용자의 브라우저 내에서만 처리되며 외부 서버로 전송되지 않습니다.

## 제3자 제공

본 확장 프로그램은 어떠한 사용자 데이터도 제3자에게 제공하지 않습니다.

## 보안

사용자 데이터는 외부 서버에 저장되지 않으며 모든 처리는 로컬 환경에서 수행됩니다.

## 정책 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있으며 변경 시 본 페이지에 공지합니다.
